The Base Starting Project for Unreal Engine is in the file 6a545180aa374be6a93d77c680327b72.zip.

Download the project zipped file. Extract it. Put the folder where you are loading your Unreal Engine project files.
Change the name of the game file "BaseStartingProject.uproject" to the desired name of your project. Make sure it has the .uproject extension.
Open Unreal Engine. Load the selected project. 


The jumping animations for the UE 4 skeletal mesh are in the TpAnims.zip file.
